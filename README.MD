# Blockchain ND Project 4 - Private Blockchain Notary Service

## Introduciton

This project provides a star registry service that allows users to claim ownership of a star. Base on a RESTful API and a private blockchain, users can lookup registered stars in the blockchain, claim a new star's ownership using his/her blockchain identity, and share a story of the claimed star.

## Deployment

This program relies on `node` 9.6+ and `npm` package manager.

```shell
npm install
node app.js
```

After deployment, API service is then available at http://localhost:8000.

## Testing

A dedicated testing script is provided to ensure all API endpoints are functional according to design.

```shell
node test.js
```

## Endpoints

Endpoints are categorized into 3 categories:

- Validation: for users to validate their own identity
- Registration: register a new star entry
- Lookup: search registered records

### Validation

#### Request for validation

POST `/requestValidation`

Sample request

```shell
curl -X "POST" "http://localhost:8000/requestValidation" \
     -H 'Content-Type: application/json; charset=utf-8' \
     -d $'{
  "address": "142BDCeSGbXjWKaAnYXbMpZ6sbrSAo3DpZ"
}'
```

Sample response

```shell
{
  "address": "142BDCeSGbXjWKaAnYXbMpZ6sbrSAo3DpZ",
  "requestTimeStamp": "1532296090",
  "message": "142BDCeSGbXjWKaAnYXbMpZ6sbrSAo3DpZ:1532296090:starRegistry",
  "validationWindow": 300
}
```

#### Sign validation message

POST `/message-signature/validate`

Sample request

```shell
curl -X "POST" "http://localhost:8000/message-signature/validate" \
     -H 'Content-Type: application/json; charset=utf-8' \
     -d $'{
  "address": "142BDCeSGbXjWKaAnYXbMpZ6sbrSAo3DpZ",
  "signature": "H6ZrGrF0Y4rMGBMRT2+hHWGbThTIyhBS0dNKQRov9Yg6GgXcHxtO9GJN4nwD2yNXpnXHTWU9i+qdw5vpsooryLU="
}'
```
Sample response

``` shell
{
  "registerStar": true,
  "status": {
    "address": "142BDCeSGbXjWKaAnYXbMpZ6sbrSAo3DpZ",
    "requestTimeStamp": "1532296090",

    "validationWindow": 193,
    "messageSignature": "valid"
  }

```
